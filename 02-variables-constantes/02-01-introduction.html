JS - II 01 Introduction


1- Qu'est-ce qu'une variable ?

Une variable est un conteneur servant à stocker des informations de manière temporaire, du texte (une chaîne de caractère) ou un nombre. La variable varie ! Elle stock différentes valeurs au fil du temps .

Une variable et la valeur sont deux choses différentes

2- Règle de déclatation

On dit (comme en PHP) que l'on "declare" une variable. On déclare une variable avec les mots-clefs :

  • let
  • var
  • Le nom d'une variable doit obligatoirement commancer par une lettre ou un underscore _
  • Le nom d'une variable ne dois contenir que des lettre, des chiffre et des _
  • Il n'y a pas d'espace
  • cf. introduction.js

3- Initialiser une variable

Quand on assigne une valeur à une varibale vide on dit que l'on stocke une valeur ou qu'on initialise une variable

4- Différences entre var et let

let est plus récent que var et contraignant que var mais ils ont la même fonction

  • A - La remontée ou un hoisting des variables

    Avec var on n'est pas obliger de déclarer la variable avant de la manipuler

  • B - la redéclaration des variables

    Avec var on avais le droit de déclarer plusieurs fois une même varibale en utilsant à chaque fois var (ce qui avait pour effet de modifier la valeur stocker) On ne peut pas le faire avec let

  • C - La portée des variables

    La "portée d'une variable désigne l'endroit où cette variable va pouvoir être utilisée dans le script.

    Nous reparlerons donc de cette portée des variables lorsque nous aborderons les fonctions en JavaScript

  • D - Conclusion - let ou var ?

    La syntaxe de déclaration des variables avec let correspond à la nouvelle syntaxe. La syntaxe avec var est l'ancienne syntaxe qui est vouée à disparaitre.

    Vous devriez donc aujourd'hui toujours utiliser le mot clef let pour déclarer vos variable, sachant c'est un peu contraigant

5 - Quelle utilité pour les variables

Il sera pratique de stocker différents types de données ou d'informations dans une variable.

Exemple avec la fonction prompt() on récupérera une données envoyée par un utilisateur, on fera un script qui contiendra une variable qui stockera les données quand elles seront envoyées
