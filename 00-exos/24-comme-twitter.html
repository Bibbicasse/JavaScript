Comme Twitter -

Un script qui contrôle que nous avont écrit 140 caractères et pas plus

Consignes

1

L'utilisateur peut saisir un texte de 140 caractères dans un textarea #tweetContenu.

Un paragraphe #compteur affiche les caracètres restants.

2

  1. Détecter l'évènement à chaque fois que l'utilisateur presse une touche.
  2. Dans la fonction lancée lorsque "l'event" est détecté, calculer le nombre de caractrère restants et l'afficher dans le p#compteur

Indications

  • Utiliser l'even keyup (cherchez à quoi il peut bien servir).
  • Utiliser la propriété elt.value (elt = élément) pour récupérer la valeur d'un input ou d'un textarea.
  • Utiliser .lengh pour compter le nombre de caractère d'une chaîne de caractères.
  • Utiliser elt.textContent=''; pour remplacer le contenu d'un élement HTML
  • La sélection d'élément avec querySelector - getElementById
  • Le gestionnaire d'évènement avec addEventListner
  • La manipulation du DOM avec - classList
