1 - If ... Else ... en JS

La structure conditionnelle if... else (en francais si... sinin) sera plus complète car on va exécuter un bloc de code si un test renvoie true ou un autre dans le cas contraire

If... Else

  • if(x > 1){
    alert('x contient une valeur supérieur à 1');
    }else{
    alert('x cotient une valeur inférieur à 1);
    }
