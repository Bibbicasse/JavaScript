1 - If ... Else ... if ... Else en JS

La condition if ... else if else (si ... sinin si ... sinon) est une structure conditionnelle encore plus complète puisqu'elle permet de générer et de prendre en charge autant de cas que l'on souhaite.

Nous allons pouvoir écrire autant de else if que l'on veut dans cette condition

if (x > 1 ) {
alert('x contient une valeur strictement supérieure à 1');
}else if (x == 1) {
alert ('x contient la valeur de 1');
} else {
alert('x contient une valeur strictement inférieure à 1'); }

Dès qu'un test va être validé, le JS va ignorer les test suivants. Il faut donc faire bien attention à l'odre des else if lors de l'écriture d'une condition pour obtenir le résultat souhaité
