JS - III 05 Les opérateurs logiques

Ce type d'opérateurs va nous permettre d'effectuer plusieurs comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test

1 - Présentation et utilisation des opérateurs logique

Le JS supporte trois opérateurs logiques : l'opérateur logique ET, l'opérateur logique OU, et l'opérateur logique NON

Opérateurs, Nom Opérateur, Symbole Descriptions
AND (ET) && envoie true si toutes les comparaisons sont évaluées à true, ou sinon false
OR (OU) || Renvoie true, si au moins l'une des comparaison est évaluée à true, ou sinin false
NO (NON) ! Renvoie false si une comparaison est évaluée à true, ou renvoie true dans le cas contraire
