JS - III 02 IF

1 - If ...

La condition if est l'une des plus utilisées et est également la plus simple à appréhender puisqu'elle va nous permettre d'exécuter un bloc de code SI et SEULEMENT SI le résultat vaut true

  • if(x > 1)
    {alert ('x contient une valeur supérieure à 1');}
  • if(x == y)
    {alert('x et y contiennent la même valeur');}
  • if(y)
    {alert("La valeur de 'y' est évalué à true");}

    • /!\ dans ce 3ème if : tout test d'une condition va être évalué dans un contexte booléan : donc à true à l'exeption des valeur suivantes qui vont être évaluée à false /!\.

  • Le booléen false
  • La valeur 0
  • Une chaîne de caractère vide
  • La valeur null
  • La valeur undefined
  • La valeur NaN (Not a Number)

2 - Inverser la valeur logique d'un test

Dans un if, le code placé dans le if n'est exécuté que si le résultat de la comparaison est true

Dans certaines situaton, on préfère vérifier si la comaparaison est false

On peut vérifier que c'est faux en comparant explicitement le résultat de la comparaison à false
