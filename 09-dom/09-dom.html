1 - Le DOM est une interface de programmation pour des documents HTML ou XML, qui représente le document (la page web actuelle) sous une forme qui permet aux langages de script comme le JavaScript d'y accéder et d'en manipuler le contenu et les styles.

3 - Alors que JavaScipt est le langage de programme qui permet d'opérer sur les objets DOM et de les manipuler, le DOM fournira les méthodes et les propriétés nécessaires pour lire, modifier, metttre à jour et supprimer des éléments du document examiné. Par exemple, vous pouvez récupérer la valeur d'un champs texte HTML dans une chaîne à l'aide du DOM

Exemple :

  • getElementsByTagName("balise")
  • getElementById("id")
